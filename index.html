HTML

  • HTML is a language for the elaboration of web pages that...
  • It has a solid and clear structure.
  • Handles block elements.
  • Allows linking with external files to define the style of the pages (CSS).
  • Allows linking with external programs to give functionality to the page (JacaScript).

CSS

  • CSS is a language used to style a web page that allows the user to...
  • to define paragraph sizes and typography
  • define colors and shapes for blocks
  • define classes for styles common to more than one element
  • define identifiers for elements that require a particular style

Git

  • GitHub is a tool for change and version control during the development of web pages that includes...
  • Creating repositories
  • The creation of events to be achieved
  • The use of branches for change control and change authorization
  • Publishing projects

JavaScript

  • JavaScript is a programming language that can be linked to a page built in HTML to provide functionality to the web page
  • To see the functionality, we most use the CROME dev tools facilities to inspect the page and see ther CONSOLE message.
